# config

Dotfiles repo. Works on macOS, Ubuntu Linux, and Windows WSL.

## What's Tracked

- **Neovim** — `init.lua` (plugins auto-bootstrap via lazy.nvim)
- **Zsh** — `.zshrc`, `.zprofile` (cross-platform Homebrew/Deno/NVM)
- **Git** — `.gitignore_global`
- **Claude Code** — `settings.json`, plugins, skills
- **Tmux** — `.tmux.conf`

## Setup

```bash
git clone <this-repo> ~/projects/config
cd ~/projects/config
./install.sh
```

The install script will:
1. Prompt to install missing tools (Homebrew, Oh My Zsh, SDKMAN, NVM, Neovim, Tmux)
2. Back up existing configs to `~/.dotfiles-backup/<timestamp>/`
3. Symlink everything into place

Safe to re-run — idempotent.

## Not Tracked

- `.gitconfig` — set up manually per machine (`git config --global user.name "..."`)
- `lazy-lock.json` — auto-generated by Neovim plugin manager
- SDKMAN binaries — just the init lines in `.zshrc`
